# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe (XO)** game built with **HTML, CSS, and JavaScript**.  
Players can enter their names, compete against each other, and track scores across multiple rounds.  

---

## 🚀 Features

- Two-player mode with custom player names.
- Scoreboard to keep track of wins.
- Detects **winner** or **draw** automatically.
- Highlights winning cells.
- Restart option to reset the board.
- Responsive and clean UI.

- Works smoothly on **desktop** and **mobile**.  
- Grid layout adapts for smaller screens.

---

## ⚡ How to Play

1. Enter names for **Player X** and **Player O**.  
2. Click **Done** to start the game.  
3. Players take turns clicking on the cells.  
4. The game will announce a **winner** or **draw** automatically.  
5. The board resets for the next round, and scores are updated.
